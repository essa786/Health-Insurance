4.d) Is the distribution of bmi across women with no children, one child and two children, the same?

One-Way ANOVA¶

In [19]:
H0= "Number of children has no effect on bmi" # Null Hypothesis
HA= "Number of children has an effect on bmi" # Alternative Hypothesis

n_females= copy.deepcopy(dataframe[dataframe['sex']=='female'])

zero = n_females[n_females.children == 0]['bmi']
one = n_females[n_females.children == 1]['bmi']
two = n_females[n_females.children == 2]['bmi']

f_stat, p_value= stats.f_oneway(zero,one,two)


if p_value < 0.05:
    print(f'{HA} as the p-value ({p_value.round(3)}) < 0.05')
else:
    print(f'{H0} as the p_value ({p_value.round(3)}) > 0.05')

Number of children has no effect on bmi as the p_value (0.716) > 0.05

In this example, p value is 0.716 and > 0.05 so we accept the null hypothesis.¶
